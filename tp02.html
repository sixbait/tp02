Ejercicio 1: Enlaces

1.1- ¿qué es una URL?

Es una cadena de texto que especifica donde puede encontrarce un recurso en el internet

1.2- ¿qué es una url absoluta?

Las url absulutas proporcionan la URL completa del documento

1.3- ¿qué es una url relativa?

Una url relativa describe la ruta relativa a un documento actual

1.4- ¿cuál es la diferencia entre una url relativa y una url absoluta?

La diferencia entre una URL relativa y una URL absoluta es que la url relativa no requiere el protocolo o nombre de dominio, solo el nombre de ruta
